### 1. 聚合根

#### 1.1 Bearing — 轴承聚合

轴承是系统的核心聚合根。每个轴承属于一个品牌和一个类型，可与多家商家关联。

| 属性组 | 字段 | 类型 |
|--------|------|------|
| 标识 | Id, CurrentCode(型号), FormerCode(曾用代号), CodeSource(代号来源) | Guid, string |
| 描述 | Description, StructureType, SizeSeries, IsStandard | string, bool |
| 尺寸 | Dimensions.InnerDiameter, OuterDiameter, Width | Dimensions 值对象 |
| 性能 | PerformanceParams.DynamicLoadRating, StaticLoadRating, LimitingSpeed | PerformanceParams 值对象 |
| 其他 | Weight, PrecisionGrade, Material, SealType, CageType | decimal?, string |
| 图片 | Image3D, Image2DCAD | string? |
| 来源 | DataSource.SourceType, DataSource.SourceSite | DataSource 值对象 |
| 统计 | ViewCount, SearchCount | int |
| 追溯 | OriginCountry, Trademark, Category | string? |
| 关联 | Brand(Brand), BearingType(BearingType) | 导航属性 |
| 集合 | Interchanges, MerchantBearings, FavoritedByUsers | 导航属性集合 |

**工厂方法：**
- `Bearing.CreateBearing(...)` — 创建标准轴承
- `Bearing.CreateNonStandard(...)` — 创建非标轴承

**业务方法：**
- `UpdateTechnicalSpecs(...)` — 更新技术参数
- `SetOrigin(...)` — 设置产地
- `IncrementViewCount()` — 增加查看次数
- `IncrementSearchCount()` — 增加搜索次数

**唯一约束：** `(CurrentCode, BrandId)` 联合唯一索引

#### 1.2 Merchant — 商家聚合

| 属性组 | 字段 | 说明 |
|--------|------|------|
| 标识 | Id, Name, EnglishName, CompanyName | Guid, string |
| 联系 | ContactInfo(Phone, Mobile, Email, Address, ContactPerson) | ContactInfo 值对象 |
| 经营 | BusinessScope, Description, LogoUrl, Type | string |
| 认证 | Status, VerifiedAt, ApprovedAt | 枚举 + DateTime? |
| 档次 | Grade, SortOrder | string, int |
| 集合 | Staff, MerchantBearings | 导航属性集合 |

**状态方法：** `Verify()`, `Approve()`, `Suspend()`, `Reject()`

#### 1.3 User — 用户聚合

与 Identity 服务的 OidcUser 通过 `AuthUserId` 桥接。

| 属性 | 说明 |
|------|------|
| AuthUserId | Identity 服务的用户 GUID |
| Nickname, Avatar, UserType | 业务层用户信息 |
| MembershipLevel | Normal / VIP / Admin |
| 集合 | Roles, FavoriteBearings, FollowedMerchants, BearingHistory, MerchantHistory |

**特殊能力：** `MigrateGuestData()` — 游客数据自动迁移

### 2. 实体

| 实体 | 表名 | 关键字段 | 说明 |
|------|------|------|------|
| Brand | brands | Code, Name, Level, Country, LogoUrl, Website, Description, SortOrder | 品牌字典 |
| BearingType | bearing_types | Code, Name, Description, SortOrder | 轴承类型字典 |
| BearingInterchange | bearing_interchanges | SourceBearingId, TargetBearingId, InterchangeType, Confidence, IsBidirectional | 替代品关系 |

| AuditLog | audit_logs | UserId, Action, ResourceType, ResourceId, Details | 管理员操作日志 |
| ApiCallLog | api_call_logs | Endpoint, Method, StatusCode, Duration, UserId, IpAddress | API 调用日志 |
| SystemConfig | system_configs | Key, Value(JSON) | 系统配置键值存储 |

### 3. 值对象

| 值对象 | 属性 | 说明 |
|--------|------|------|
| Dimensions | InnerDiameter, OuterDiameter, Width | 均为 decimal?，单位 mm |
| PerformanceParams | DynamicLoadRating, StaticLoadRating, LimitingSpeed, LimitingSpeedGrease, LimitingSpeedOil | 载荷单位 kN，转速单位 rpm |
| DataSource | SourceType(枚举), SourceSite(string?), SourceRecordId(string?) | Manual/Crawler/FileImport/Api |
| ContactInfo | ContactPerson, Phone, Mobile, Email, Address | EF Core OwnsOne 映射 |

### 4. 基类

**BaseEntity** — 提供 `Id(Guid)`, `CreatedAt`, `UpdatedAt`, `IsActive`，以及 `AddDomainEvent()` 领域事件支持。

**ValueObject** — 基于组件值的相等性比较，不可变模式。
